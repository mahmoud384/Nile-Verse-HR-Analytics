# HR & Recruitment Analytics Dashboard (Nile Verse Case Study)

# Project Overview
This project focuses on analyzing the Human Resources and Recruitment data for **Nile Verse**. The goal was to provide data-driven insights to optimize workforce management, monitor employee performance, and improve the recruitment funnel efficiency for the year 2024.

# Business Objectives
* Analyze **Workforce Demographics** (Headcount, Gender Diversity, and Departmental distribution).
* Evaluate **Recruitment Efficiency** (Time-to-Hire, Conversion Rates, and Drop-off Rates).
* Correlate **Performance vs. Compensation** to ensure fair pay practices.
* Provide an **Interactive Dashboard** for executives to filter data by Date, Location, and Department.

# Tech Stack & Tools
* **Power BI:** Data Visualization & Dashboarding.
* **DAX (Data Analysis Expressions):** Created measures for Turnover rates, Conversion rates, and Year-to-Date metrics.
* **Power Query:** Data cleaning, transformation, and handling missing values.
* **Microsoft Excel:** Initial data source and exploratory analysis.

##  Key Insights & Results
* **Recruitment Funnel:** Identified the specific stages where the highest percentage of candidates "drop off," allowing the TA team to fix the bottleneck.
* **Hiring Channels:** Discovered which recruitment platforms provide the fastest "Time-to-Hire."
* **Pay Equity:** Visualized the relationship between Performance Ratings and Total Pay across different cities.
